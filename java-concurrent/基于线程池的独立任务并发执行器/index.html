    【Java】基于线程池的独立任务并发执行器

    发表于 2022-06-11 阅读次数：

    目的：

    对于多个独立的任务，可以以并发的方式执行任务，以提高 CPU 利用率，提高处理效率。

    思路

    在一个线程池中，开启指定数量的线程，每个线程从任务队列中获取任务执行。

    执行的过程中，判断当前线程是否在执行任务的状态，如果没有执行任务，取一条任务执行，如果正在执行，则跳过，下轮再判断。

    在所有任务执行完后，关闭线程池。

    需要注意的是数据结构的选择，须选择并发类的数据结构，不然可能出现阻塞，死锁等情况。
