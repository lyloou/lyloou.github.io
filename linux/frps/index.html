frps 是一个优秀的内网穿透软件。

frps 下载安装：https://github.com/fatedier/frp/releases

服务端: ./frps -c ./frps.ini

frps.ini

客户端: ./frpc -c ./frpc.ini

frpc.ini

访问服务器端的 dashboard 查看统计信息: http://frps.example.com:7500/

访问内网 ssh： ssh -oPort=7022 test@x.x.x.x
通过浏览器访问内网机器: http://office.frps.example.com:2780/
