在 github 提交 pr

  • fork 一份到自己的仓库
  • 设置本地的 origin url: git remote set-url origin https://github.com/lyloou/WeexErosFramework.git

  • 进入自己 fork 的项目

  • 有个new pull request按钮，按照提示进行即可
