## 表格
```markdown
| name       | age |
| ---------- | --- |
| LearnShare | 12  |
| Mike       | 32  |
```

| name       | age  |
| :--------- | :--- |
| LearnShare | 12   |
| Mike       | 32   |

- 第二行的破折线，至少需要3个。


## 实现换行
在一段文字的后面添加两个空格  
自动就换行了。
